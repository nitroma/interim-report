According to Froment/Bischoff, all 4 reactors chosen (packed-bed microreactor, trickle bed reactor, xxx, xxx) can be assumed to have the same [froment bischoff], which can be defined in the equation below:

The packed-bed microreactor and trickle bed reactor are assumed to have the same design equation as a plug flow reactor, using the design equation is defined as below:

\begin{equation}
    V_R = \int_{F_{A0}}^{F_{A}} \frac{dF_A}{-r_A} = \int_{F_{A0}}^{F_{A}} \frac{dF_A}{-kC_A}
    \label{reactor_sizing}
\end{equation}

Packed bed microreactors are assumed to model plug flow behaviour diameter of the catalyst is < 0.1 times the inner diameter of the reactor.
